🐙 GitHub Configuration

Your PAT is stored only in your browser's localStorage. It never leaves your device except to authenticate with GitHub.

How to Get a GitHub Personal Access Token
Step-by-step instructions:
  1. Go to github.com → Settings → Developer Settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Give it a name like "Mission Site Admin"
  4. Set expiration to No expiration (or 1 year)
  5. Check the "repo" scope (full control of private repos)
  6. Click Generate token and copy it
  7. Paste it in the field above and click Save
