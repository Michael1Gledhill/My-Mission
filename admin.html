Token stored only in your browser's localStorage. Never sent anywhere except to GitHub's API.

How to Get a Personal Access Token
  1. Go to github.com → Settings → Developer Settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Name: Mission Site Admin — Expiration: No expiration
  4. Check the "repo" scope (top checkbox under repo)
  5. Click Generate token — copy it immediately
  6. Paste it above and click Save GitHub Config
